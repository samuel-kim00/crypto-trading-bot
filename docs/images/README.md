# Dashboard Screenshots

This directory contains screenshots and images for documentation.

## Dashboard Screenshot

Add a screenshot of the dashboard (`dashboard.png`) to help users visualize the system.

To capture:
1. Run the dashboard: `python3 src/dashboard/app.py`
2. Open http://localhost:8081 in your browser
3. Take a screenshot of the main dashboard view
4. Save as `dashboard.png` in this directory
